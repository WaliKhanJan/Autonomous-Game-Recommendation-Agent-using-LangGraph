
# Autonomous Game Recommendation Agent using LangGraph

Built an autonomous game recommendation agent using LangGraph, based on probabilistic decision-making, state management, and graph-based workflow orchestration.



## Installation

1. Recommended Python 3.13 or earlier 


2. Install dependencies

```bash
pip install -r requirements.txt
```



## How to Run

1. Run the .py file by selecting the Python Kernal

2. After execution, the workflow visualization will be saved as:

```bash
langgraph.png
```
